# 📊 Customer Churn Prediction in E-Commerce Platforms

## 📌 Project Title

Customer Churn Prediction in E-Commerce Platforms

---

## 📖 Overview

This project aims to predict customer churn in e-commerce platforms using machine learning techniques.
By analyzing customer behavior and transaction data, the model identifies patterns that indicate whether a customer is likely to leave the platform.

---

## 🚀 Run Instructions

Clone the repository:

```bash
git clone https://github.com/khoan2202/STAT3013.Q21_GroupUseYourMind.git
cd STAT3013.Q21_GroupUseYourMind
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Launch Jupyter Notebook:

```bash
jupyter notebook
```

Open and run:

```
notebooks/modal.ipynb
```

> Ensure the dataset file is in the correct location.
